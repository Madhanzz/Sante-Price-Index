# Sante Price Index 🌿📊

Sante Price Index is an Android application developed using Kotlin and Firebase that helps users and small vendors track real-time mandi prices and calculate profitable selling prices. The application provides a clean and modern interface for monitoring market prices, viewing trends, and calculating margins easily.

---

## 🚀 Features

* 📈 Real-time mandi price updates
* 💰 Profit and selling price calculator
* 📊 Price Watch and Price Board screens
* 🔥 Firebase Realtime Database integration
* 🌐 Backend automation using Node.js
* 📱 Modern UI built with Jetpack Compose
* 🔄 Automatic data synchronization

---

## 🛠 Technologies Used

* Kotlin
* Jetpack Compose
* Firebase Realtime Database
* Node.js
* data.gov.in API
* Android Studio

---

## ⚙️ How It Works

1. A backend Node.js script fetches mandi price data from the data.gov.in API.
2. The script updates Firebase Realtime Database automatically.
3. The Android application reads live data from Firebase.
4. Users can calculate selling prices using transport cost and profit percentage.

---

## 📱 Modules

* **Price Watch** – Displays live mandi prices
* **Price Board** – Market-style display screen
* **Profit Calculator** – Calculates recommended selling prices
* **Trends** – Shows price-related insights

---

## 🎯 Project Objective

The main objective of this project is to help users and small vendors monitor market prices and make better pricing decisions using real-time data and automated updates.
